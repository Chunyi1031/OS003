# OS003 源代码

## 源代码编译环境
|项目      |要求                     |
|---------|-------------------------|
|操作系统  |Linux（建议Ubuntu/Debian） |
|编译器    |GNU-GCC(x86_64-linux-gnu-gcc)|
|Make     |GNU-Make                 |
|虚拟机    |qemu-system-x86_64       |

## 运行要求
|项目      |要求                     |
|---------|-------------------------|
|架构      |x86_64                   |
|CPU      |单核                      |
|内存      |256MB                    |
|硬盘      |128MB                    |

## 编译
- 编译`make`
- 制作虚拟硬盘`make disk`
- 复制文件到虚拟硬盘`make system`
- 运行(自动更新虚拟硬盘)`make run`
- 清理(不清理虚拟硬盘)`make clean`
- 创建系统目录 `make esp`
- 测试 `make test`
- 创建未创建的目录`make fix`
- 创建vmdk`make vmdk`

## 命令
| 命令 | 功能描述 |
|------|----------|
| `clear` | 清屏命令，清除当前屏幕显示内容 |
| `status` | 显示系统状态信息，包括操作系统名称、CPU频率、屏幕尺寸、字体地址、ACPI RSDP地址和总内存大小 |
| `shutdown` | 关机命令，延迟100毫秒后关闭系统 |
| `reboot` | 重启命令，延迟100毫秒后重启系统 |
| `time` | 显示当前时间（年/月/日 时:分:秒） |
| `clock` | 启动时钟界面，持续显示实时时间直到按E键退出 |
| `UEFImd` | 显示第6个UEFI内存描述符信息和总内存大小 |
| `pmm` | 显示物理内存管理器信息，包括内存描述符地址、数量和详细信息 |
| `acpmm` | 测试物理内存分配器功能，执行分配和释放操作 |
| `StartTest` | 创建并启动测试线程，在屏幕上绘制变化的彩色方块 |
| `KillTest` | 终止测试线程 |
| `ThreadLs` | 显示当前运行的所有线程 |
| `vmm` | 查看虚拟内存信息 |
| `acvmm` | 分配虚拟内存并尝试写入数据后回收虚拟内存 |
| `uvmth` | 创建使用虚拟内存为栈的线程 |
| `uvltest` | 运行一个简单的用户态线程，运行带syscall的测试代码 |
| `disk` | 扫描所有硬盘，并打印基础信息 |
| `fat32info` | 显示FAT32分区信息 |
| `findpart` | 找到第一个ATA控制器的主盘的第ESP分区，并打印所在位置和分区名 |
| `readfile` | 读取文件`Test.txt`内容 |
| `writefile` | 覆盖文件`Test.txt`内容（最后复原） |
| `appendfile` | 在文件`Test.txt`末尾追加内容（不复原） |
| `ls` | 显示根目录下的内容 |
| `rename` | 修改文件`Test.txt`的文件名为`ABC.TXT`（最后复原） |
| `touch` | 创建一个文件并删除 |
| `mkdir` | 创建一个空目录并删除 |
| `gpu` | 显示显卡信息 |
| `gpumode` | 置显卡模式（切换分辨率为1920x1080） |
| `blit` | 位图传输测试 |
